### Theory Questions

**Q1: Unit vs Integration test?**  
A: Unit: isolated. Integration: multiple components.

**Q2: Mock purpose?**  
A: Replace real dependency with controlled fake.

**Q3: pytest vs unittest?**  
A: pytest: simpler assert, fixtures. unittest: built-in.
